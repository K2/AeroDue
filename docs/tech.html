Architecture

Four modules. One pipeline.

:core
Pure Kotlin. DOT × EU261 × card/business perks. No Android deps. Source of truth for dollar amounts — the LLM never invents money.
:llm
LiteRT-LM 0.13. LlmRationaleRunner + CPU backend. Qwen2.5-0.5B-Instruct bundle (~9s gen on Pixel 8). Rules fallback if missing.
:app
Jetpack Compose / M3. Assistant, filing coordinator, notifications, consent store, Integrations screen for MCP connectors.
backend/
Python host. CLI + YAML corpus + pytest parity with core. Not bundled in the APK.
GPS / matcher→ CompensationEngine→ Receipt UI→ FilingCoordinator

Gotchas we locked in
Don't regress these
  • Bundle must include LlmMetadata
  • Non-empty start_token (Qwen: <|im_start|>)
  • GPU aborts — use CPU only
  • Native SIGABRT can't be caught in Kotlin
fallback chain · AssistantService
// cloud connector (opt-in) → on-device LiteRT → deterministic templates
connectors?.generate(prompt, system) ?: llm.generate(prompt, system) ?: ""
